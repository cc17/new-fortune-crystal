# 🔮 水晶球占卜 - Fortune Crystal

一个基于 React 的占卜 Web 应用，包含塔罗牌占卜和生日占卜两个功能模块。

## ✨ 功能特性

### 页面一：塔罗牌占卜
- 点击抽取随机塔罗牌
- 动画效果展示抽卡过程
- 使用柱状图展示运势分析（爱情、财富、事业、学业）
- 可重新抽取

### 页面二：生日占卜
- 输入出生日期（年月日）
- 选择想咨询的问题类型（爱情、事业、财富、学业）
- 输入邮箱地址
- 自动计算生肖和星座
- 展示占卜答案（当前为 Mock 数据，后续可接入豆包大模型）
- 推荐商品列表展示

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后访问控制台显示的本地地址（通常是 http://localhost:5173）

### 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **React 18** - UI 框架
- **Vite** - 构建工具
- **React Router Dom** - 路由管理
- **Recharts** - 数据可视化（柱状图）

## 📂 项目结构

